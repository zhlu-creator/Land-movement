# Land-movement

## 利用GNSS速度场确定青藏高原活动地块运动与应变参数

技术栈：GNSS速度场数据、最小二乘平差、协方差传播率、MATLAB编程（矩阵运算、权阵构建、参数估计）

具体工作：构建并实现了五种地壳运动与应变模型（模型一至五），涵盖平移、正应变、剪应变、刚性旋转等参数；基于1375个GPS站点的速度场数据（东向、北向速度及标准差），建立1494×1494权阵，采用最小二乘法求解各模型参数；对比不同块体中心点选择对参数估计的影响，并分析各模型的适用性与优劣。

量化成果：模型一至四求解出6个参数（如平移速率、应变率、旋转率），模型五求解3个旋转参数；模型二在应变和旋转参数的方差上优于模型一，量级改善约 十的负十八次方；所有参数统一至国际单位制（m/yr、rad/yr），并给出协方差阵评估精度。
